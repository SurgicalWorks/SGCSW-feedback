# SGCSW Feedback

Repositorio público para reportar incidencias, bugs y feedback de SGCSW.

## Qué reportar aquí

- Errores visibles en la aplicación.
- Problemas de acceso o comportamiento inesperado.
- Sugerencias de mejora funcional.
- Dudas de uso que puedan convertirse en mejora de producto.

## Qué NO publicar

Por seguridad y privacidad, **no incluyas nunca**:

- Datos personales.
- Datos clínicos o información de pacientes.
- Credenciales, contraseñas, tokens o claves.
- Capturas con información sensible visible.
- URLs privadas, datos contractuales o información interna.

Si una incidencia requiere información sensible, escribe una descripción general y contacta por el canal privado acordado.

## Cómo reportar un bug

Abre una issue usando la plantilla **Bug report** e incluye:

1. Qué estabas intentando hacer.
2. Qué ocurrió.
3. Qué esperabas que ocurriera.
4. Pasos para reproducirlo.
5. Navegador/dispositivo si aplica.
6. Captura solo si no contiene información sensible.

## Nota

Este repositorio no contiene código fuente. El código privado de SGCSW permanece separado.
